Django


(a) What is Django and what is its main purpose?

Django is a Python-based web application framework
Django's main purpose is to simplify and speed up the development of web applications by providing a set of tools and abstractions that allow developers to focus on writing their application's logic instead of dealing with low-level details such as handling HTTP requests and responses, database integration, URL routing, security, and more.


(b) What is the difference between a view and a template in Django?

View is responsible for processing the user's request and returning a response.
While template is responsible for defining the structure and layout of the HTML output for that response.



(c) What is a model in Django and how is it used?

Model is a class that represents a database table and is used to define the data structure of the app.
We use models to design tables (data structures) to get, store and access data.
